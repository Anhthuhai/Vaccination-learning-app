Vaccination Learning App

Scaffolded Expo (React Native) project for a mobile learning app about vaccines.

Features scaffolded:
- Thông tin về các loại vaccine
- Lịch tiêm chủng
- Quiz / bài tập kiểm tra kiến thức
- Tình huống lâm sàng
- Thông tin an toàn và tác dụng phụ
- Hồ sơ người dùng

Quick start

1. Install dependencies (requires Node.js and npm/yarn, and Expo CLI if you prefer globally):

```bash
cd /Users/ngocanh/development/Vaccination-learning-app
npm install
# or
# yarn install
```

2. Start the app with Expo:

```bash
npm run start
# then press 'a' to open Android emulator/device, 'i' for iOS (macOS), or scan the QR code with Expo Go
```
